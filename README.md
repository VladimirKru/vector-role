vector-role v.1.0.0
=========

This role install Vector.

Role Variables
--------------
You can change the version of the program:

    vector_version: "0.30.0"

Example Playbook
----------------

    - hosts: all
      roles:
        - role: vector-role
            version: "0.30.0"
